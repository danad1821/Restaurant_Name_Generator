# Restaurant Name Generator #
## Overview ##
This is a simple Streamlit application that uses the Gemini 2.0 Flash model via the LangChain framework to generate a fancy restaurant name and a suggested list of 10 menu items based on a user-selected cuisine.

## Technologies Used ##
* Python
* Streamlit
* LangChain
* Google Gemini API

## Example Usage ##
* Open the app: `https://restaurantnamegenerator-jhqprk3un7267vfkpgyjy9.streamlit.app/`.
* Use the sidebar dropdown to select a cuisine (e.g., 'French').
* The application will instantly display the generated Restaurant Name and a list of Menu Items.
* Click the "Regenerate" button to get a new suggestion for the current cuisine.
